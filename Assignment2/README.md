# Gemini Price Scout

A Chrome extension for real-time price comparison across major Indian e-commerce platforms using Google Gemini AI.

## Setup
1. Load as an unpacked extension in Chrome from `chrome://extensions`.
2. Ensure your `.env` key is correctly applied (already done in `popup.js`).

## Platforms Supported
- Amazon.in
- Flipkart
- Myntra
- Nykaa

## Tech Stack
- Manifest V3
- Gemini 2.5 Flash API
- Vanilla JavaScript / CSS / HTML
